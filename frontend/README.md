# BacklogOS — Frontend (React JSX + Vite)

Plain React, no TypeScript. Marketing site, demo login, full dashboard, profile, admin.

## Run

```bash
npm install
npm run dev
```

Opens on http://localhost:5173. Backend on :5000 is optional — app works fully offline.

## Demo accounts

- User: `demo@backlogos.app` / `demo`
- Admin: `admin@backlogos.app` / `admin`

## Routes

- `/` landing
- `/features`, `/pricing`, `/about`, `/contact`
- `/login`
- `/app` dashboard (auth required)
- `/profile` (auth required)
- `/admin` (admin only)
